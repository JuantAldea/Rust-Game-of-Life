# Rust-Game of Life
Conway's Game of Life implemented in Rust

## Compile & Run
```$ cargo run --release```
## Controls
* `scape`: exit
* `<space>`: pause/unpause
* `C/c`: clear world
* `R/r`: generate random world
* `T/t`: tick world
